## f2b-manager 项目方案设计

为用户设计了 VPS Fail2ban 管理系统的详细方案，产出了完整设计文档 `设计方案.md`。

### 核心需求
1. fail2ban 安装/卸载/更新集成
2. Telegram 机器人定时推送 IP 封禁情况
3. Telegram 实时预警

### 方案要点
- **技术栈**: Python 3.10+ / python-telegram-bot / APScheduler / SQLite / systemd
- **架构**: 单守护进程 (f2b-manager.service) 内集成 4 大模块
  - Fail2ban 管理模块 (installer/manager/config_builder)
  - Telegram Bot (命令交互 + 三级鉴权)
  - Scheduler (定时报告 + 健康检查)
  - Notifier (实时预警)
- **实时预警关键设计**: Hook + 轮询双保险
  - Hook: fail2ban 自定义 action (telegram-notify.conf) → notify.sh → f2b-manager CLI，毫秒级实时
  - 轮询兜底: Scheduler 每 5 分钟对比 SQLite 状态库，补发遗漏事件
- **安全**: chat_id 白名单三级权限、配置文件 600、危险操作二次确认、消息限流去重
- **部署**: 一键 install.sh + systemd 服务
